# Phylogenetic and biochemical analysis of the evolution and interdomain modularity of bifunctional L-fucokinase/GDP-fucose pyrophosphorylases

Smith NT, Hodgins AJ, Boddington ME, Capicciotti CJ, diCenzo GC, Howe GW. 2025. **Phylogenetic and biochemical analysis of the evolution and interdomain modularity of bifunctional L-fucokinase/GDP-fucose pyrophosphorylases**. *Journal of Biological Chemistry*. 301(12): 110937.

This repository contains the scripts required to repeat the analyses reported in the associated article.
